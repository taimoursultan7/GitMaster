# ⚡ GitHub Commands

## Initialize Git

```bash
git init
```

## Add Files

```bash
git add .
```

## Commit Files

```bash
git commit -m "First Commit"
```

## Connect GitHub Repository

```bash
git remote add origin https://github.com/username/repository.git
```

## Push Code

```bash
git push -u origin main
```
